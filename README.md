# 💧 WaterTech OS - Aqlli Sug'orish Tizimi

**WaterTech OS** — bu qishloq xo'jaligi va uy bog'dorchiligi uchun mo'ljallangan, real vaqt rejimida ishlovchi aqlli monitoring va boshqaruv paneli.

## ✨ Asosiy Imkoniyatlar
* **Real vaqt monitoringi:** Tuproq namligi, havo harorati va namligini kuzatish.
* **Masofaviy boshqaruv:** Suv nasoslarini (Plot 1, Plot 2) va yoritish tizimini masofadan yoqish/o'chirish.
* **Vizual grafiklar:** Datchiklardan kelayotgan ma'lumotlarni dinamik grafiklar orqali tahlil qilish.
* **Aqlli Jadval:** Sug'orish vaqtini sekundigacha aniqlikda rejalashtirish.

## 🛠 Texnologiyalar
* **Frontend:** React.js, Tailwind CSS
* **Ikonkalar:** Lucide-React
* **Grafiklar:** Recharts
* **Hardware (Apparat qismi):** ESP32, DHT11 (Havo datchigi), Soil Moisture Sensor (Tuproq datchigi)
